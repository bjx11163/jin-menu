✂️ 裁剪工具
自由裁剪 / 预设比例(1:1/4:3/16:9) / 旋转 / 翻转
🎨 滤镜
5-8个预设滤镜：暖食/复古/清新/黑白/胶片/美食增强
✨ 调色
亮度/对比度/饱和度/色温 — 滑条调节
✏️ 加文字
输入菜名/描述，选字体/颜色/大小，拖拽摆放位置
💡 实现方案：用微信小程序 canvas 2D API 实现编辑功能。图片先上传到云存储临时区，编辑完成后再保存到正式路径。所有操作在手机端完成，不需要后端处理图片。
