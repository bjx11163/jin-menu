# 私人点菜小程序 — 设计文档

## 概述

私人点菜小程序，供我和女朋友两人使用。她扫码进入小程序，从我的拿手菜菜单中点菜下单，我收到通知后去做菜，做完标记完成，她可以评价。整个系统使用微信小程序 + 微信云开发，无需自建服务器。

## 角色

| 角色 | 身份 | 主要操作 |
|------|------|---------|
| 厨师（我） | 管理员 | 菜品管理、接单、统计、互动 |
| 顾客（她） | 普通用户 | 看菜单、点菜、评价、个人中心 |

厨师入口通过首次设置简单密码保护，防止误入。

## 核心流程

1. 她扫码进入小程序 → 看到菜单
2. 浏览菜品（分类浏览/搜索/猜你想吃/随机选）
3. 选菜加入购物车（或单点直接提交）
4. 填写备注 + 可选预约时间 → 提交订单
5. 订单写入云数据库 → 云函数发送微信订阅消息通知我
6. 我打开管理端看到待处理订单 → 去做菜
7. 做完标记"已完成"（可选上传成品照）
8. 她看到订单完成 → 评价打分

## 功能清单

### 女朋友端 — 3个底部Tab

**Tab 1: 菜单首页**
- 顶部大背景图（复古街景，毛玻璃效果）
- 搜索框（按菜名搜索）
- 分类标签横向滚动（荤菜/素菜/汤/凉菜/主食）
- 菜品列表卡片（照片/菜名/我的评分/已点次数/上次吃时间/收藏按钮）
- 右下角浮动按钮 — 随机选菜
- 今日推荐卡片 — 猜你想吃（根据点单习惯+天气推荐）
- 投票入口 — 我发起的二选一投票

**Tab 2: 购物车/点单**
- 已选菜品列表（数量+-/删除）
- 留言备注输入框
- 预约时间选择
- 提交订单按钮

**Tab 3: 个人中心**
- 点单历史（按时间倒序，显示状态）
- 我的评价（打分记录）
- 我的收藏（收藏的菜，可快速加购）
- 心愿菜单（"我想吃这个，你快学"）
- 你做过的菜（我的成品照片合集）
- 纪念日进度（在一起天数）

### 管理端（我）

**首页概览**
- 待处理订单列表（她点了啥/预约几点/备注）
- 一键标记"已做好"
- 今日接单数 + 本月累计

**菜品管理**
- 新增菜品（填菜名/分类/描述/用料/标签/难度/评分）
- 上传照片 → 弹出编辑器（裁剪+滤镜+加文字）
- 编辑/删除已有菜品
- 管理分类（增删改排序）

**图片编辑器**
- 裁剪（自由/1:1/4:3/16:9）+ 旋转 + 翻转
- 滤镜（5-8个预设：暖食/复古/清新/黑白/胶片/美食增强）
- 调色（亮度/对比度/饱和度/色温滑条）
- 加文字（菜名水印，拖拽位置，选字体颜色大小）
- 基于微信小程序 Canvas 2D API 实现

**统计页面**
- 月度做饭日历/折线图
- 最受欢迎菜 Top 10
- 点单趋势分析
- 她的打分汇总

**互动管理**
- 心愿菜单 — 看她提议了啥，标记"已学"或"安排"
- 发起投票 — 编辑两道菜让她二选一
- 纪念日设置 — 设定日期，到点触发特餐提示

## 数据库设计（云数据库）

### dishes（菜品）
- `_id, name, photo, category, description`
- `ingredients[]` 用料列表
- `difficulty` 难度
- `rating` 我的评分
- `tags[]` 标签（辣度/口味等）
- `createdAt, updatedAt`

### categories（分类）
- `_id, name, sortOrder`

### orders（订单）
- `_id, userId, items[{dishId, name, quantity, note}]`
- `scheduledTime` 预约时间
- `status` pending / done
- `createdAt, completedAt`

### reviews（评价）
- `_id, orderId, dishId, rating(1-5), comment, photo, createdAt`

### favorites（收藏）
- `_id, userId, dishId`

### wishlist（心愿）
- `_id, userId, dishName, status` 提议/已学/安排

### votes（投票）
- `_id, createdBy, options[], expiresAt, result`

## 技术方案

| 层 | 技术 | 说明 |
|---|------|------|
| 前端 | 微信小程序原生 | wxml + wxss + js |
| 后端 | 微信云开发 | 云数据库 + 云存储 + 云函数 |
| 推送 | 微信订阅消息 | 她首次授权后，下单时通知我 |
| 图片处理 | Canvas 2D API | 手机端本地处理，不上传服务器 |

**页面清单（约10页）：**
1. 启动页（角色选择）
2. 菜单首页
3. 菜品详情
4. 购物车
5. 她的个人中心
6. 管理首页（接单）
7. 菜品编辑
8. 图片编辑器
9. 统计页面
10. 互动管理（心愿/投票/纪念日）

## 视觉风格

- 背景：用户提供的复古欧洲街景图（全屏，半透明毛玻璃覆盖保证文字可读）
- 主色调：暖棕 #8B6B4D / 复古金 #D4AF37 / 深咖 #3E2723
- 卡片：毛玻璃效果（backdrop-filter: blur）
- 圆角：12px-16px 大圆角

## 开发估算

- 核心功能（菜单浏览 → 点菜下单 → 接单标记 → 通知）：约1周
- 完整功能（含统计/图片编辑/互动）：约1-2周
- 一个人每天2-3小时的话：约2-3周

## 部署流程

1. 注册微信小程序（个人主体，免费）
2. 微信开发者工具中开通云开发
3. 开发完成后提交微信审核（1-2天）
4. 生成小程序码，她扫码即可使用
